# 15.1 地址空间
### 内存区域：
可以被访问的合法地址空间
```
进程只能访问有效内存区域内的内存地址
各种内存对象
.text
.data
.bss
用户空间栈
共享库
堆
```

### 内存描述符
```c++
struct mm_struct
{
    struct vm_area_struct *mmap; /*内存区域链表*/
    struct rb_root mm_rb; /*VMA形成的红黑树*/
    pgd_t *pgd; /*页全局目录*/
    atomic_t mm_users; /*使用地址空间的线程数 */
    atomic_t mm_count; /*使用地址空间的进程数 */
    struct list_head mmlist; /*所有mm_struct形成的链表*/
}
```
